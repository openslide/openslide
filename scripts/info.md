# openslide/scripts

## But
Scripts de build et distribution pour OpenSlide.

## Pourquoi
Automatise la creation de packages et l'installation des dependances.

## Comment
- `dist.py` : script de packaging pour distribution
- `vcpkg-install.sh` : installation des dependances via vcpkg (Windows)

## Structure
```
scripts/
  dist.py            # Packaging distribution
  vcpkg-install.sh   # Installation deps Windows via vcpkg
```
